Aseprite Scripts

Export as PSD

psdを出力するスクリプト。 Luaでフルスクラッチでpsd出力書くのかなり辛かった記憶がある。 そもそもpsdのフォーマットを詳しく解説してる文書が少なくて、 Adobe Photoshop File Formats Specification は網羅的に書いてあるけど説明不足だし、なかなか情報がなくて大変だった。

Asepriteが公式にpsdの対応をするという話が出ているのだけれどまだ開発段階らしい。 Asepriteの場合現状LuaのスクリプトからCのDLLを呼び出せないのでネイティブで実装してくれるなら それに越したことはないよね。DLL呼び出しは開発者の人に聞いたら難しいらしい、、、。 ただちょっと寂しいよね。色んな人が使ってくれているらしいし。

Smooth Filter

ドット絵の角を滑らかにするといい感じになるというのを知って作ったフィルタ。 正直aseprite上で実装するより単独のアプリにするとかWebAppにしたほうが良かったと思う。

Lcd Pixel Filter

各色をRGB成分に分解して並べるスクリプト。 RGBパターンを変えたりすると結構楽しい。 二色刷りみたいなこともできて楽しい。 三原色だけじゃなくて任意の(RGB空間で一次独立なベクトルを持つ)色でやっても面白いかもしれない (表現できない色が出てくるけどもともと色変わっちゃうし面白さはあるよね)。
